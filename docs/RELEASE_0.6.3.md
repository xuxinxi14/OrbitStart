# OrbitStart 0.6.3

## 更新内容

- 新增悬浮球功能：主窗口隐藏后可通过桌面悬浮球快速回到 OrbitStart。
- 悬浮球使用 Local Galaxy 风格贴图，鼠标悬停时会从无光晕状态切换为有光晕状态。
- 悬浮球支持拖动和靠边吸附，左键点击恢复主窗口，右键或鼠标悬停可打开快捷菜单。
- 快捷菜单以独立桌面窗口展示，包含搜索、添加资源、工作区、最近项目和设置入口。
- 悬浮球设置已整理到设置中心的独立页面，可控制是否启用、尺寸、透明度、置顶、贴边和悬停展开行为。
- Windows 下悬浮球收起态使用系统圆形窗口区域，展开菜单使用独立胶囊窗口。
