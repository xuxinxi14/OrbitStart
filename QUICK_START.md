## 🚨 注意事项

### 如果端口被占用
```bash
# 方法 1：更换端口
npm run dev -- --port 1421

# 方法 2：查找并关闭占用端口的进程（Windows）
netstat -ano | findstr :1420
taskkill /PID [进程ID] /F
```

### 如果样式未生效
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 硬刷新页面（Ctrl+F5）
3. 检查开发者工具 Console 是否有错误

## 🌌 设计哲学

> OrbitStart 不是一个启动器，而是一个本地资源星系的导航控制台。每一个应用、文件、网址都是星系中的资源节点，用户是星际旅行者，通过这个控制台探索和跳跃到各个资源点。

**视觉隐喻**：复古未来主义 + 深空探索 + 航海仪器 + 星际导航
