# OrbitStart Local Galaxy 设计升级总结

## 📋 设计目标

将 OrbitStart 从普通 Web UI 风格升级为「深邃、复古、唯美、丝滑」的 Local Galaxy 星际风格。

## ✨ 主要改进

### 1. 文案艺术化升级

#### 品牌标识
- **顶部标识**：`Local Galaxy` → `LOCAL GALAXY`（全大写，更具品牌力）
- **品牌副标题**：`Local Galaxy` 保持优雅的混合大小写

#### 核心术语系统化
| 原文案 | 新文案 | 设计理念 |
|--------|--------|----------|
| 启动工作台 | 本地星系 | 更符合 Local Galaxy 主题 |
| 设置 | 轨道控制 | 航天主题隐喻 |
| 运行日志 | 星际日志 | 增强太空感 |
| 资产实验室 | 设计星图 | 更诗意的表达 |
| 插件 | 引擎 | 统一为航天术语 |
| 工作台 | 星系导航 | 强化导航概念 |

#### 界面文案优化
- **资源库** → **资源星图**
- **个收藏** → **颗星标**（使用量词"颗"更有星空感）
- **总条目** → **总节点**（网络化、星系化）
- **快捷导航** → **快速跃迁**（科幻感）
- **扫描桌面/开始菜单** → **扫描本地程序**（更简洁）
- **导入 Edge/Chrome 书签** → **导入浏览器书签**（泛化表达）
- **扫描 Obsidian Vault** → **连接知识库**（更有科技感）
- **导出 JSON 备份** → **导出数据备份**（更通用）

#### 状态信息艺术化
- **星际状态** → **轨道状态**
- **核心模块、插件、主题和本地索引已就绪** → **本地星系运行正常 · 所有引擎已就绪**
- **资源总数** → **资源节点**
- **已启用插件** → **活跃引擎**
- **ON/OFF** → **启用/关闭**（中文化）

#### 副标题诗意化
| 页面 | 新副标题 |
|------|----------|
| 本地星系 | 星际资源导航站 · 探索你的个人宇宙 |
| 轨道控制 | 插件引擎 · 主题工作室 · 数据星图 |
| 星际日志 | 追踪每一次资源跳跃与插件脉冲 |
| 设计星图 | 素材预览 · 混合模式 · 视觉调试 |

#### 英文标签优化
- `Plugin Manager` → `引擎管理器`
- `Acceptance` → `实时响应`
- `Plugin Dev Kit` → `开发套件`
- `Theme Dev` → `主题开发`
- `Local Paths` → `本地优先`
- `Plugin Logs` → `引擎日志`
- `General` → `基础设置`
- `Launcher` → `启动器`
- `Theme Studio` → `主题工作室`
- `Shareable` → `可分享`

### 2. 字体排版优化

#### 字间距（Letter Spacing）系统
```css
/* 全局文本 */
letter-spacing: 0.015em;  /* 增加呼吸感 */

/* 大标题 */
.topbar h1 {
  letter-spacing: 0.02em;  /* 标题更宽松 */
}

/* 副标题 */
.title-subtitle {
  letter-spacing: 0.03em;  /* 副标题最宽松，营造飘逸感 */
}

/* EYEBROW 标签 */
.eyebrow {
  letter-spacing: 0.08em;  /* 全大写需要更大间距 */
}

/* 品牌名称 */
.brand-mark strong {
  letter-spacing: 0.05em;  /* 品牌标识清晰度 */
}

.brand-mark span {
  letter-spacing: 0.12em;  /* "Local Galaxy" 副标题飘逸 */
}

/* 主按钮 */
.primary-action {
  letter-spacing: 0.02em;  /* 操作按钮易读 */
}
```

#### 字重（Font Weight）调整
```css
/* 品牌副标题 */
.brand-mark span {
  font-weight: 600;  /* 增加品牌副标题的分量 */
}

/* 大标题 */
.topbar h1 {
  font-weight: 700;  /* 从默认提升到 700 */
}

/* 章节标题 */
.section-head h2 {
  font-weight: 600;  /* 适度的权重感 */
}
```

#### 字号微调
```css
/* 主标题放大 */
.topbar h1 {
  font-size: 34px;  /* 从 31px 提升 */
}

/* 品牌名称 */
.brand-mark strong {
  font-size: 14px;  /* 从 13px 提升 */
}

/* 空状态标题 */
.empty-state strong {
  font-size: 18px;  /* 明确定义 */
}

/* 状态块文本 */
.status-block strong {
  font-size: 15px;  /* 明确定义 */
}
```

## 🎨 设计原则总结

1. **深邃**：深空背景 + 柔和星云 + 细微星点
2. **复古**：衬线标题字体 + 暗金色系 + 雷达/星盘装饰
3. **唯美**：诗意文案 + 柔和光晕 + 飘逸字间距
4. **丝滑**：160ms 过渡 + 轻微上浮 + 背景模糊
